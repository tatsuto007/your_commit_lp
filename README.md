# your_commit_lp

`your_commit` のホームページ（LP）専用リポジトリです。  
React + Vite + TypeScript で構成されています。

## セットアップ

```bash
npm install
```

## 開発サーバー起動

```bash
npm run dev
```

デフォルトの確認URLは `http://localhost:5173` です。

## ビルド

```bash
npm run build
```

## プレビュー

```bash
npm run preview
```

## ディレクトリ構成

- `src/MarketingPage.tsx`: LP本体
- `src/styles/marketing.css`: LPスタイル
- `src/main.tsx`: エントリーポイント
